Title: Classes

Date: 9/26/14

Ruby has Object Oriented Programming. Classes may be defined: "class XYZ". Methods may be made to work with that class. The methods are defined within the body of that class. (see code below for an example). Usually, and initialize method is created that allows creation of a new instance object of that class. The object may be created by typing "XYZ.new(any parameters needed)". The method may be called by typing the object name.the method: "XYZObject.method". Instance variables ("@variablename") may be used. These variables may be used within methods of the class for a particular instance of the class without needing to pass the variable into each method as a parametere. See some example code of a class, class methods, and instance variables below.

class Die
def initialize(labels)
@labels=labels
raise ArgumentError.new("You must enter an array of labels for each side of the die.") if (@labels==[])
end
def sides
p @labels.length
end
def roll
p @labels.sample
end
end
