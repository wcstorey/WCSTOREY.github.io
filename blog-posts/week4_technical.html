Title:Enumerable#map

Date: 9/21/14

The map method pertaining to an Enumerable class may either take a single parameter ({"cat"}) or a parameter and block ({ |i| i*i }). If a parameter/block is given, then map will return a new array containing the results of running the block on each element of the focused array ((1..4).map { |i| i*i } #=> [1, 4, 9, 16]). If only the parameter is given, a new array is returned placing the parameter in the place of each element ((1..4).map { "cat" } #=> ["cat", "cat", "cat", "cat"]).
