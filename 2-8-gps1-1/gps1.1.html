Pair Programming

As defined by Bill & Lorena

What is Pair Programming?

Pair programming is when developers pair together to develop code.

Why is Pair Programming important to DBC?

Pair programming is important here at DBC because it is becoming a common practice nowadays in the industry and DBC wants its sutdents student to be prepared to enter the industry. Additionally, it helps students learn and reinforce the techniques used and better understand the rationale behind using those techniques. Further, it help develop soft skills for interacting with other developers.

Compare Driver & Navigator

The driver is the one at the keyboard typing in code listening to the direction of the navigator. The navigator is not typing, but rather oversees the direction the code is going and directs the driver what he should be programming.
